# 一、核心主题（Core Idea）

**中文：**
玫瑰产业是一个高度全球化的价值链，从品种研发到零售销售，每个环节都在创造价值，但**价值分配不均**，不同参与者的盈利能力差异巨大。

**English：**
The rose industry is a highly globalized value chain where value is created at multiple stages, but **value capture is uneven across stakeholders**.

# 二、案例主线（Emma 的困境）

### 情境：

* Emma经营花店 **Petal & Grace**
* 进价 $1.5 / 支，情人节卖 $5 / 支 → 高利润

### 三个战略选项：

**小型高端零售商如何在竞争和规模压力中生存？**

---

# 三、玫瑰产业价值链（重点）

## 1️⃣ Breeders（育种商）

**做什么：**

* 开发新品种（7–10年）
* 控制知识产权（PBR）

**价值来源：**

* 专利 + 品种控制
* 向种植者收取许可费（royalties）

**特点：**

* 技术壁垒高
* 市场集中（德国公司占全球约50%）

👉 **价值捕获：高（High value capture）**

---

## 2️⃣ Growers（种植者）

**做什么：**

* 在肯尼亚、哥伦比亚、厄瓜多尔种植
* 收获、分级、包装

**优势：**

* 低劳动力成本
* 气候优势（减少温室成本）

**劣势：**

* 资本密集（数百万美元投入）
* 利润受价格波动影响

👉 **价值捕获：中等（Medium）**

---

## 3️⃣ Logistics（物流）

**关键点：**

* 温度：2–4°C
* 湿度：80–90%
* 时间极其关键（“time is money”）

👉 作用：

* 保证品质
* 防止价值损失

👉 **价值捕获：功能性（Support role）**

---

## 4️⃣ Auctions（拍卖市场）

**特点：**

* 荷兰拍卖（Dutch auction）
* 全球最大：Royal FloraHolland

**功能：**

* 定价机制
* 市场信息
* 风险控制（保证付款）

👉 **价值捕获：稳定但有限**

---

## 5️⃣ Wholesalers（批发商）

**做什么：**

* 批量采购 + 分销
* 提供市场信息

👉 **价值捕获：中等**

---

## 6️⃣ Retailers（零售商）⭐关键

**类型：**

* 花店（高端）
* 超市（低价）

**价值来源：**

* **品牌 + 设计 +体验**
* 情感溢价（尤其情人节）

👉 Emma案例说明：

* 成本 $1.5 → 售价 $5
* 利润最高环节之一

👉 **价值捕获：非常高（High）**

---

# 四、产业关键洞察（Key Insights）

## 1️⃣ 谁最赚钱？

👉 **两端最赚钱：**

* 上游：Breeders（技术+专利）
* 下游：Retailers（品牌+情感溢价）

👉 中间环节（种植、物流）利润较薄

---

## 2️⃣ 全球分工逻辑

| 环节 | 地区      |
| -- | ------- |
| 育种 | 欧洲      |
| 种植 | 非洲 / 南美 |
| 交易 | 荷兰      |
| 消费 | 欧洲 / 全球 |

👉 典型**全球价值链分工（Global Value Chain Fragmentation）**

---

## 3️⃣ 成本 vs 溢价

* 非洲玫瑰 → 低价、大众市场
* 南美玫瑰 → 高端、长茎、奢侈感

👉 **产品定位决定价值**

---

## 4️⃣ 时间敏感性

* 花是**极易腐商品（perishable）**
* 延误 = 价值下降

---

# 五、Emma 的战略思考（Case Question）

核心问题其实是：

### 👉 她应该在哪个“价值环节”竞争？

| 选项   | 本质          |
| ---- | ----------- |
| 卖给超市 | 放弃高利润，换稳定   |
| 自营   | 高利润但风险大     |
| 扩张   | 规模化复制“品牌溢价” |

---

# 六、一句话总结（非常重要）

👉 **玫瑰产业的本质：**
**价值不在“种花”，而在“控制品种”和“接近消费者”。**
